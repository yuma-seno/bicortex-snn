# Phase 1.4: パブロフ型条件付け 検証レポート

**実施日:** 2026/01/28
**担当:** 開発チーム
**状態:** ✅ 完了 (Success)

## 1. 目的
BC-SNNアーキテクチャにおいて、以下の「連想記憶」メカニズムが正常に動作することを検証する。
1.  **文脈の統合:** 過去の文脈（ベル音）と現在の刺激（エサ）を時間差を超えて結びつけられるか。
2.  **想起 (Recall):** 学習後、文脈（ベル）のみで概念（エサ）を想起し、行動（唾液）を誘発できるか。
3.  **自然な沈静化:** 外部刺激が消失した後、無限ループに陥らず自然に活動が沈静化するか。

## 2. 最終構成 (The Golden Configuration)
試行錯誤の結果、以下の構成で安定した学習と想起が確認された。

### アーキテクチャ特性
* **Interface (Concept $\leftrightarrow$ Memory):** **固定 (Fixed)**
    * 無限ループを防ぐため、書き込み（Injection）は弱く、読み出し（Recall）も適度に抑える非対称構成を採用。
* **Internal (Memory $\to$ Memory):** **可塑的 (Plastic)**
    * SRG (意味的共鳴ゲーティング) により、記憶野内部の配線のみを書き換えて連想を形成する。

### ゴールデンパラメータ
| パラメータ | 値 | 役割 |
| :--- | :--- | :--- |
| `learning_rate` | **0.001** | トレース変数の蓄積を考慮し、非常に低く設定して重み爆発を防ぐ。 |
| `adaptation_step` | **0.3** | **最重要。** 発火ごとに閾値を上げ、刺激消失後の自己ループを断ち切るブレーキ役。 |
| `gate_ratio` | **0.15** | 不応期(2ms)による発火率上限(0.5)を考慮し、確実にゲートが開く閾値(0.4相当)を設定。 |
| `w_max_clip` | **0.8** | 内部結合の上限を制限し、単一ニューロンでの暴走を防ぐ。 |
| `w_injection` | **0.6** | 概念 $\to$ 記憶への書き込み強度。 |
| `w_recall` | **0.12** | 記憶 $\to$ 概念への読み出し強度。複数ニューロンの同期が必要な絶妙なライン。 |

## 3. 実験結果
* **学習フェーズ:** 重みが `0.0` から `约0.3` まで徐々に成長。
* **テストフェーズ:** ベル音のみの入力に対し、記憶野が活性化し、Interface経由でエサの概念が発火。
* **沈静化:** 入力がなくなると、Adaptationの効果により閾値が上昇し、ループ利得が1.0を割り込んで活動が停止した。

## 4. 考察・教訓
1.  **ループゲインの制御:** ConceptとMemoryの間のループゲインが1.0を超えると「幻覚（Lingering）」が止まらなくなる。これを防ぐには結合強度を下げるだけではなく、**「順応（Adaptation）」による動的なゲイン抑制**が極めて有効である。
2.  **スケール感:** トレース変数($\tau=2000$)は数百〜数千まで蓄積するため、学習率は $1/1000$ オーダーでなければならない。
3.  **不応期の罠:** ニューロンが物理的に発火できない期間を考慮して、ゲート閾値を設定する必要がある。
